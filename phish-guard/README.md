# Phish Guard - Chrome Extension

Phish Guard is a student-built Chrome extension focused on safe browsing and ethical cybersecurity learning. The block list is curated by the developer and delivered through extension updates, with plans to expand detection and automation over time.

## Features
- Developer-managed block list for known phishing domains
- Simple on-page block message for blocked domains
- Local-only storage (no external requests)
- Enable/disable toggle and basic statistics

## Installation
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `phish-guard` folder

## Usage
1. Open the extension settings
2. Review the current blocked list
3. Any matching domain or subdomain is blocked

## Project Goals
This extension is built as a student project to learn secure browsing concepts, extension development, and responsible security practices.

## Privacy
All data is stored locally. No tracking and no external requests.
