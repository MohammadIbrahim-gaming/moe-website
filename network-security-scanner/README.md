# Network Security Scanner (Python)

## Overview
This Python-based network security scanner is a student project for educational and defensive cybersecurity use. It discovers active devices, identifies open ports, and highlights exposed services to support learning and risk awareness.

## Ethical Use and Legal Disclaimer
This tool is intended only for:
- Networks you own, or
- Systems where you have explicit written permission to test

Unauthorized scanning is illegal and unethical. By using this tool, you agree to:
- Follow all applicable laws and regulations
- Use it strictly for ethical learning and defensive security
- Take full responsibility for your usage

The author is not responsible for misuse.

## Purpose
This project was built to:
- Learn how network reconnaissance works at a protocol level
- Understand how attackers identify attack surfaces
- Apply ethical hacking principles in a defensive context
