Network Security Scanner (Python)
Overview

This project is a Python-based network security scanner designed for educational and defensive cybersecurity purposes. It allows users to discover active devices, identify open ports, and analyze exposed services in order to better understand and reduce potential security risks.

This tool was built to demonstrate networking, TCP/IP fundamentals, and ethical security practices.
