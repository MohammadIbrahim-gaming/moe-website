# Mohammad Ibrahim - Personal Website

A professional portfolio site built as a student project to showcase cybersecurity work and personal projects.

## Highlights
- Home page with terminal-style bio
- Projects page featuring Phish Guard, Network Security Scanner, and Party Arena
- Party Arena: 2-player local multiplayer game with 20 mini-games

## Tech Stack
- HTML5
- CSS3 (animations and visual effects)
- JavaScript (ES6 modules)
- Terminal-style UI design

## About
I am a Computer Science student at the University of New Orleans with a cybersecurity focus. This site is a student-built portfolio to document learning, projects, and ethical security work.
